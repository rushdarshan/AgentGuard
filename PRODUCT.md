# Product

## Register

product

## Users

Hackathon judges and attendees evaluating the AgentGuard tool live in a demo setting. They need to grasp the security narrative quickly — what was tested, what was caught, what got through — without side narration. Hallway-track visibility and immediate readability are requirements.

## Product Purpose

AgentGuard is a CI/CD-style adversarial testing harness that systematically red-teams AI agents against real-world attack vectors: prompt injection, tool-argument manipulation, cross-tenant data leaks, and Unicode encoding bypasses. It produces a composite reliability score with a pass/fail gate. The dashboard visualizes these test results, execution traces, and impact summaries so viewers can instantly assess agent security posture.

## Brand Personality

Confident and clinical. Security-first, precise, authoritative. The tone of a threat detection console — no unnecessary drama because the data is already dramatic enough. Commands respect through precision, not through theatrical effects.

## Anti-references

- Cyberpunk / gaming aesthetic: neon-on-black, glitch effects, excessive glow effects. These undermine clinical credibility.
- Generic SaaS dashboard templates: overused sidebar nav patterns, gradient headers, cookie-cutter card grids.

## Design Principles

1. **Confidence through precision**: Every number, score, and label carries weight. This is security data — no ambiguity, no decoration that undermines trust.
2. **Demo-readiness**: The interface must work on a big screen in a hallway track setting — readable from distance, self-explanatory without narration.
3. **Clinical over theatrical**: Security doesn't need neon glitch effects to feel important. Clean hierarchy, authoritative typography, color that signals without shouting.
4. **Show the story**: The trace graph and impact summaries should tell the attack narrative at a glance, not hide in tooltips and expandable panels.

## Accessibility & Inclusion

Relaxed — this is a hackathon demo tool. Standard color contrast is observed where practical, but the primary viewing context is a large demo screen with a presenter narrating.
