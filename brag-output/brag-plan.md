# Brag Plan: AgentGuard

## What is this app?
AgentGuard is a brutal pre-deployment stress testing harness for AI agents that intercepts runtime traffic to catch prompt injection, data leaks, and jailbreaks.

## The angle
The video plays on the anxiety of shipping AI blindly. It starts with the realization that your agent is unprotected, then slams into the solution: AgentGuard's cascade graph and multi-model judge. Tone is serious, cinematic, and intimidating.

## Hook (first 2-3 seconds)
"You're deploying your AI agent without a security test."
(Text types out like a terminal command in JetBrains Mono. Sudden red flash.)

## Key moments (the middle)
- "Intercepts runtime traffic. No code changes." (A code snippet of the proxy starts running, terminal green on black).
- "10 attack categories. Multi-model judge." (A barrage of attack types flashes by quickly: Prompt Injection, Data Leak, Jailbreak, etc.)
- "Visualize failure cascades." (A force-directed graph node structure builds itself, nodes turning red as vulnerabilities are found.)

## Outro / punchline
"Stop shipping blindly." 
(AgentGuard logo/icon ⚡ hits the screen). "AgentGuard — CI for AI Agents."

## User flow worth showing
Test endpoint -> See cascade graph of vulnerabilities -> Blocks push if below threshold (Pre-push hook).

## Tone
- Preset: cinematic
- Creative direction: Aggressive cybersecurity tool / Pre-deployment stress test.
- Interpretation: Fast cuts, stark contrast (red/green on black), dramatic tension, authoritative typography.

## Format: landscape — 1920x1080
## Duration: 20 seconds

## Visual identity (from the project)
- Background: #0A0A0A
- Accent: #E61919 (red)
- Text: #EAEAEA (phosphor)
- Terminal Green: #4AF626
- Display font: Inter
- Body font: JetBrains Mono
- Strongest visual element: The terminal proxy and the neo4j cascade graph.

## Share copy (draft)
Stop shipping your AI agents blindly. AgentGuard runs 10 attack categories and visualizes failure cascades before you hit production. ⚡

## Audio direction
- Role: cinematic support
- Music: Cinematic bed, dramatic, serious.
- Music treatment: Low swell building up, hard stops on the red flash.
- SFX posture: dense; motion-matched terminal typing, heavy impacts, digital glitches.
- Restraint rule: Do not use cheerful or upbeat sounds; keep it tense and professional.

## Storyboard

### Scene 1 — The Problem — 4s
Black screen. JetBrains Mono text types out: "Deploying your AI agent without a security test?"
Sequential/interaction: Yes — simulate terminal typing.
Audio intent: Tense, digital.
Audio-coupled idea: type the hook with terminal key ticks.
Transition mood: Hard cut / Red flash → Scene 2

### Scene 2 — The Solution / Proxy — 4s
Fast zoom into a mock terminal. "Intercept runtime traffic. No code changes." Green text flows down.
Sequential/interaction: Yes — text scrolling down fast.
Audio intent: High-tech, data processing.
Transition mood: Glitch cut → Scene 3

### Scene 3 — The Attacks — 5s
"10 attack categories. Multi-model judge." A grid of cards flashes into view, one by one: Prompt Injection, Jailbreak, Data Leak.
Sequential/interaction: Yes — cards pop in sequentially.
Audio-coupled idea: Heavy card impacts on each beat.
Transition mood: Slide wipe → Scene 4

### Scene 4 — Cascade Graph — 4s
"Visualize failure cascades." A dark background where red and green nodes connect in a force-directed graph layout.
Sequential/interaction: Nodes pulsing.
Audio intent: Ethereal but tense.
Transition mood: Quick fade → Scene 5

### Scene 5 — Outro — 3s
"Stop shipping blindly. AgentGuard." The ⚡ icon lands with weight.
Audio intent: Final heavy impact, then ringing silence.
